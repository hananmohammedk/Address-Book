# Address Book – C Programming Project

## 📌 Description
The Address Book project is a **menu-driven contact management application**
implemented in **C**.  
It allows users to **store, view, search, update, and delete contact details**
using persistent file storage.

This project focuses on **modular programming, file handling, and structured
data management**, which are fundamental concepts for system-level and embedded
software development.

---

## 🧠 Concepts Used
- Structures
- File handling (text / CSV files)
- Pointers
- Modular programming
- Menu-driven application design
